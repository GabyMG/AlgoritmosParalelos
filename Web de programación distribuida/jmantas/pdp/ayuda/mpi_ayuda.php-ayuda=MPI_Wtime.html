MPI_Wtime

Descripción breve

Devuelve el tiempo trascurrido hasta el momento de la llamda.

Sintaxis

En C 

#include <mpi.h>
double MPI_Wtime()

En C++ 

#include <mpi.h>
double MPI::Wtime()

Descripción

MPI_Wtime devuelve numero de segundos trasncurdos desde el comienzo del proceso. Los tiempos son locales al nodo que los llama. No hay requisito de que los diferentes nodos deban tener el mismo tiempo.

Notas

Esta llamada es frecuentemente usada para tomar tiempos de duracion de un algoritmo.

double Inicio, Fin;
Inicio = MPI_Wtime();
{
	..... Ejecucion del algoritmo .....
}
Fin = MPI_Wtime();
cout << Tiempo de ejecucion :<< Fin-Inicio<< segundos <<endl;
