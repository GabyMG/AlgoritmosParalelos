Instalaciones

OpenMPI Descarga

OpenMPI es una implementación muy eficiente de la interfaz de paso de mensajes (MPI). Esta proporciona un compilador integrado con GCC en linux para C y C++. En este manual te guiaremos paso a paso para realizar la instalación de dicha herramienta.

Web oficial: www.open-mpi.org

Eclipse Descarga

Eclipse es un IDE de programación creado y especializado en Java. A través de plugins es capaz de tomar diversas funcionalidades. En este manual te mostraremos cómo instalar el IDE paso a paso y a instalar los plugins necesarios para programar para C y C++ y para poder depurar código paralelo.

Web oficial: www.eclipse.org

Herramientas de análisis y mejora (TAU) Descarga

TAU (Tuning and Analysis Utilities) es una herramienta para visualizar y analizar las acciones que realiza un programa paralelo durante su ejecución. Esta herramienta permite analizar código en C y C++. En este manual te guiaremos paso a paso para realizar la instalación de dicha herramienta.

Web oficial: www.cs.uoregon.edu/research/tau

Herramientas de visualización gráfica MPE Descarga

MPE (MPI Parallel Environment) es una herramienta de análisis para la ejecución de programas paralelos con MPI. Esta herramienta únicamente es capaz de analizar (compilar y ejecutar) código escrito en C, razón por la cual damos este material a título informativo en esta asignatura.

Documentación

Interfaz de Paso de Mensajes (MPI) Descarga

MPI ("Message Passing Interface", Interfaz de Paso de Mensajes) es un estándar que define la sintaxis y la semántica de las funciones contenidas en una biblioteca de paso de mensajes diseñada para ser usada en programas que exploten la existencia de múltiples procesadores.

Web oficial: www.mcs.anl.gov

Más información en: www.open-mpi.org/doc/v1.4/

OpenMP Descarga

OpenMP es un modelo portable y escalable con una interfaz simple y flexible para el desarrollo de aplicaciones paralelas en plataformas que van desde ordenadores de sobremesa hasta supercomputadoras.

Web oficial: openmp.org
